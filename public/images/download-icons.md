# 앱 아이콘 다운로드 가이드

다음 앱들의 공식 아이콘을 다운로드해주세요:

1. **카카오맵**
   - Google Play: https://play.google.com/store/apps/details?id=net.daum.android.map
   - 파일명: kakao-map.png

2. **네이버 지도**
   - Google Play: https://play.google.com/store/apps/details?id=com.nhn.android.nmap
   - 파일명: naver-map.png

3. **티맵**
   - Google Play: https://play.google.com/store/apps/details?id=com.skt.tmap.ku
   - 파일명: tmap.png

## 다운로드 방법

1. 각 링크를 브라우저에서 열기
2. 앱 아이콘 이미지에서 마우스 우클릭
3. "이미지를 다른 이름으로 저장"
4. 위의 파일명으로 저장
5. `/public/images/` 폴더에 저장

또는 브라우저 개발자 도구에서:
1. F12를 눌러 개발자 도구 열기
2. Network 탭 선택
3. 페이지 새로고침
4. 이미지 파일 필터링
5. 앱 아이콘 이미지 URL 찾기 (보통 512x512 크기)
