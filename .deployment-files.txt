## GitHub Actions 워크플로우 (4개)

1. .github/workflows/deploy.yml
   - 메인 배포 워크플로우
   - main 브랜치 push 시 자동 실행
   - Vite 빌드 + HTML 최적화 + GitHub Pages 배포
   - 실행 시간: 약 2-3분

2. .github/workflows/preview.yml
   - Pull Request 프리뷰
   - 빌드 검증 및 크기 리포트
   - PR 코멘트 자동 생성

3. .github/workflows/performance-check.yml
   - 성능 체크 워크플로우
   - Lighthouse CI 실행
   - 빌드 메트릭 측정

4. .github/workflows/lighthouse.yml
   - Lighthouse CI 상세 분석
   - Core Web Vitals 측정
   - 성능 점수 리포트

## 빌드 스크립트 (2개)

1. build.sh
   - 로컬 프로덕션 빌드
   - Vite 빌드 + 최적화
   - 빌드 크기 요약 출력
   - 실행: ./build.sh

2. optimize.sh
   - 이미지 최적화 스크립트
   - PNG/JPEG 압축 + WebP 변환
   - 최적화 전후 비교
   - 실행: ./optimize.sh

## 설정 파일 업데이트

   - terser 의존성 추가
   - devDependencies에 포함

   - base: './' (상대 경로)
   - minify: 'terser'
   - drop_console: true
